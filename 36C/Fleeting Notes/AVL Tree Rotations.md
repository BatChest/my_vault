# AVL Tree Rotations
In order to maintain the balnce of an AVL tree, we must re-balance which is done using rotations.
There are Four unique rotations for re-balancing
1. Right Rotation
2. Left Rotation
3. Left-Right Rotaion
4. Right-Left Rotation
